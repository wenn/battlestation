# Battlestation
Setup a fresh work machine with [ansible][ansible].

[Credit to Josheph Kahn's blog][josephkahn].

## Macos
Setup your macos machine with ansible.

#### Requirements
1. [homebrew][brew]

#### Setup
`./macos_run.sh`

#### Supports

| Installation | Version | Can be disabled? |
| --- |---:|---:|
| git | latest | no |
| python | 2.7 | no |
| ansible | 2.4.3.0 | no |
| chrome | latest | yes |
| vim | latest | yes |
| tmux | latest | yes |
